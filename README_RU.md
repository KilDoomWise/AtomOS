# Atom OS
**Легковесная микроядерная операционная система для OpenComputers (Minecraft)**

> Atom OS — это минималистичная операционная система, созданная с упором на чистую архитектуру, низкое потребление ресурсов и удобство (User-Friendly). В основе лежит модульный подход, строгое разделение привилегий и отказ от глобального разделения состояния в пользу изолированных сред.

> По производительности есть большая разница, в простое OpenOS активно использует 350-400 КБ памяти, Atom использует всего лишь 135 КБ активно, это минимальный объем, а это значит что Atom способен работать исправно на самом слабом компьютере который можно собрать в моде!

## Особенности архитектуры
* **Микроядро и уровни привилегий (Rings):** Архитектура системы построена на кольцах защиты. 
  * **Ring-1:** Системные драйверы (управление файловой системой, GPU, прерываниями).
  * **Ring-2:** API-библиотеки, предоставляющие безопасные обертки.
  * **Ring-3:** Пользовательская песочница. Пользовательские программы не имеют прямого доступа к таблице `component`. Все вызовы к железу проходят через защищенный мост (`unit.lua`).
* **Модульная оболочка (MES):** Командная строка не вшита в ядро ОС. MES (Modular Executing Shell) — это обычное пользовательское приложение, расположенное в `/Apps/MES.app`. Команды оболочки (`ls`, `cd`, `open` и т.д.) не хардкодятся, а динамически загружаются из `/Libraries/MES/`. Вы можете легко написать и использовать собственный шелл вместо стандартного.
* **Инкапсуляция приложений:** Программы не разбросаны по файловой системе в виде одиночных `.lua` скриптов. Они хранятся и исполняются как директории-пакеты `.app` (например, `App.app/main.lua`), что сохраняет корень системы в чистоте.
* **Планировщик задач (APS):** Многозадачность реализована на базе асинхронных корутин с поддержкой уровней приоритета. Встроенный диспетчер задач (`htop`) позволяет мониторить и завершать процессы. 
  * *Примечание:* Многозадачность является кооперативной. Если программа уходит в бесконечный цикл без вызова `coroutine.yield()` (или обработки системных событий), она заблокирует всю систему.
* **Ultimate Edit:** Встроенный продвинутый редактор кода. Включает нативную подсветку синтаксиса Lua, полную поддержку мыши (скролл, перемещение курсора кликом), горячие клавиши для поиска и навигации, а также оптимизированный кэш отрисовки строк.
* **Userspace:** в Atom есть полноценная система пользователей, разделение на Root и пользовательские папки

## Установка

1. Вставьте дискету OpenOS (Operating System), она по умолчанию есть в моде
2. Также обязательно требуется интернет-карта! Поскольку мы будем устанавливать Atom через интернет
3. Запустите компьютер вместе с ней и дождитесь загрузки, после введите команду ниже (чтобы вставить надо нажать на колесико мыши)
```
wget -q https://raw.githubusercontent.com/KilDoomWise/AtomOS/refs/heads/main/Installer/openos.lua /tmp/installer.lua && /tmp/installer.lua
```
4. После запустится автоматизированный установщик, следуйте инструкциям которые он предлагает

Готово!
