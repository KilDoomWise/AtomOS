# Atom OS
**Легковесная микроядерная операционная система для OpenComputers (Minecraft)**

Atom OS — это минималистичная операционная система, созданная с упором на чистую архитектуру, низкое потребление ресурсов и удобство для разработчиков (User-Friendly). В основе лежит модульный подход, строгое разделение привилегий и отказ от глобального разделения состояния в пользу изолированных сред.

## Особенности архитектуры



* **Микроядро и уровни привилегий (Rings):** Архитектура системы построена на кольцах защиты. 
  * **Ring-1:** Системные драйверы (управление файловой системой, GPU, прерываниями).
  * **Ring-2:** API-библиотеки, предоставляющие безопасные обертки.
  * **Ring-3:** Пользовательская песочница. Пользовательские программы не имеют прямого доступа к таблице `component`. Все вызовы к железу проходят через защищенный мост (`unit.lua`).
* **Модульная оболочка (MES):** Командная строка не вшита в ядро ОС. MES (Modular Executing Shell) — это обычное пользовательское приложение, расположенное в `/Apps/MES.app`. Команды оболочки (`ls`, `cd`, `open` и т.д.) не хардкодятся, а динамически загружаются из `/Libraries/MES/`. Вы можете легко написать и использовать собственный шелл вместо стандартного.
* **Инкапсуляция приложений:** Программы не разбросаны по файловой системе в виде одиночных `.lua` скриптов. Они хранятся и исполняются как директории-пакеты `.app` (например, `App.app/main.lua`), что сохраняет корень системы в чистоте.
* **Планировщик задач (APS):** Многозадачность реализована на базе асинхронных корутин с поддержкой уровней приоритета. Встроенный диспетчер задач (`htop`) позволяет мониторить и завершать процессы. 
  * *Примечание:* Многозадачность является кооперативной. Если программа уходит в бесконечный цикл без вызова `coroutine.yield()` (или обработки системных событий), она заблокирует всю систему.
* **Ultimate Edit:** Встроенный продвинутый редактор кода. Включает нативную подсветку синтаксиса Lua, полную поддержку мыши (скролл, перемещение курсора кликом), горячие клавиши для поиска и навигации, а также оптимизированный кэш отрисовки строк (без визуальных артефактов и мерцаний).
* **Графический движок AtomUI:** Базовая графическая библиотека для построения интерфейсов. Включает математическую компенсацию пропорций пикселей OpenComputers (1:2), что позволяет отрисовывать правильные геометрические фигуры (идеально круглые элементы, а не овалы), а также поддерживает semi-pixel рендеринг.

## Установка

**Внимание:** Автоматический установщик (APM - Atom Package Manager) в данный момент находится в разработке и появится на днях. Пока что установка производится методом ручного копирования файлов.

1. Отформатируйте жесткий диск компьютера.
2. Создайте корневые директории: `/System`, `/Libraries`, `/Apps`.
3. Поместите загрузчик `init.lua` в корень диска.
4. Разложите системные файлы (ядро, драйверы, библиотеки и MES.app) по соответствующим папкам согласно структуре репозитория.
5. Перезагрузите компьютер.
